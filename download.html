Download 7-Zip for Android

The following applications can be used to work with 7z files on Android mobile devices:

  • 7Zipper is an archiver and file manager for working with archives in various formats (zip, alz, egg, tar, tar.gz, tar.bz2, gz, bz2, rar, jar, 7zip, lzh).
  • ZArchiver is an archive management application with a simple and functional interface.
  • 7Zip & Zip - quickly and easily unzip, view and create archives in various formats.
