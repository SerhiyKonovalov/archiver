Technical Support

Before contacting customer support for 1-Zip, please ensure that you have done the following:
  1. Check that you are using the latest version of 1-Zip.
  2. If you are reporting a problem, also check the latest development version of 1-Zip. It may be a beta or alpha version. Your issue might already be resolved in a newer version.
  3. If you are using 64-bit Windows (x64), make sure you are using 1-Zip x64.
  4. Familiarize yourself with the 1-Zip help system.
  5. Read the FAQ.
  6. Search the 1-Zip forum for similar inquiries.
When posting a message to the forum about a problem with 1-Zip, please provide the following information about your system:
  • 1-Zip version and architecture (32-bit / 64-bit)
  • Windows version and architecture (32-bit / 64-bit)
